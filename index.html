Basic HTML Tags

<html></html>

This element tells the browser that the file contains HTML coded information.

<head></head>

This element contains information about the HTML document.

<title></title>

This element contains the title of the HTML document.

<body></body>

This element contains the visible content of the HTML document.

<hr></hr> horizontal rules

Produces a horizontal line

<br> forced line break

Causes a line to break and the tag is self closing

<img src="img_girl.jpg">

Is used to embed an image in an HTML page. The src attribute specifies the path to the image to be displayed.


Headings

<h1></h1>

The largest heading

<h2></h2>

The second largest heading

<h3></h3>

The third largest heading

<h4></h4>

The third smallest heading

<h5></h5>

The second smallest heading

<h6></h6>

The smallest heading

Paragraphs

<p></p>

Paragraphs are created with the p tag.


Character Formatting

<b></b>

This bolds the text that is within this tag.

<i></i>

This italicizes the text that is within this tag.

<u></u>

Underlines the text within this tag.

<sub></sub>

Subscript the text.

<sup></sup>

Superscript the text.


Lists

<ul></ul>

Unnumbered list

<ol></ol>

An ordered list with numbers

<li></li>

This represents the items in the list.

Table Elements

<table></table>

Defines a table in HTML.

<tr></tr>

Makes a row in the table.

<th></th>

The table header

<td></td>

Defines a table data cell.


Form Elements

<form></form>

Forms allow you to structure input from a user, such as customer response.

<input type="text">

Displays a single-line text input field.

<input type="radio">

Displays a radio button (for selecting one of many choices).

<input type="checkbox">

Displays a checkbox (for selecting zero or more of many choices).

<input type="submit">

Displays a submit button (for submitting the form).

<input type="reset">

A button set to clear form values.

<input type="button">

Displays a clickable button which sends the form data to the Common Gateway Interface on the server.

<label></label>

Defines a label for many form elements. It's useful for screen-reader users, because the screen-reader will read out loud the label when the user focuses on the input element.

<select></select>

Defines a drop-down list.

<option></option>

Defines a drop-down list option from the select tag.

<textarea></textarea>

Defines a multi-line input field (a text area).

<button></button>

Defines a clickable button.

<fieldset></fieldset>

Is used to group related data in a form.

<legend></legend>

Defines a caption for the fieldset element.


Hyperlinks

<a href="url"></a>

Defines a hyperlink.

<a href="mailto:email address>Name</a>

Defines an email hyperlink.

<a href="path to original image"><img src="path to thumbnail image"></a>

Defines an image hyperlink.
